description=One-click Relative Elevation Model (detrended DEM) for any river in the current map view. Downloads the highest-available OpenTopography DEM, builds a REM via RiverREM (with a native scipy IDW fallback), and loads it styled.
about=Zoom the QGIS canvas to any river, click the button, and get a styled Relative Elevation Model. The plugin reads the current canvas extent, transforms it to an EPSG:4326 bounding box, downloads the finest DEM OpenTopography offers for that area (USGS 3DEP 1m/10m/30m in North America, otherwise COP30/SRTMGL3), reprojects it to the local UTM zone, derives a river centerline (OpenStreetMap via Overpass -> GRASS hydrology -> a manually selected line layer), and detrends the DEM into a REM. RiverREM is the primary engine; a native scipy.spatial KDTree IDW fallback runs automatically when RiverREM is unavailable. Needs an OpenTopography API key (set it in River REM Settings).
